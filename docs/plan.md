## Υπερπαραμετροι

| Παραμετρος | Προεπιλογη | Περιγραφη |
|------------|------------|-----------|
| epochs | 500 | Μεγιστες εποχες εκπαιδευσης |
| learning_rate | 0.001 | Ρυθμος μαθησης |
| momentum | 0.9 | Ορμη SGD |
| batch_size | 16 | Μεγεθος batch |
| hidden_sizes | [64, 32, 16] | Νευρωνες ανα κρυφο επιπεδο |
| dropout | 0.3 | Ρυθμος dropout |
| weight_decay | 1e-4 | L2 κανονικοποιηση |
| early_stopping | 30 | Υπομονη πρωιμης διακοπης |

## Αναμενομενα Αποτελεσματα

- Ακριβεια δοκιμης: ~60-70%
- Καλη γενικευση (μικρο κενο train-val)
- Ολοκληρωμενες οπτικοποιησεις
- Λειτουργικο CLI

## Σημειωσεις

- Το συνολο δεδομενων ειναι μικρο (297 δειγματα μετα τον καθαρισμο)
- 5-class classification ειναι πιο δυσκολο απο binary
- Η κατανομη κλασεων ειναι ανισορροπη
- Χρησιμοποιειται weighted averaging για τις μετρικες
