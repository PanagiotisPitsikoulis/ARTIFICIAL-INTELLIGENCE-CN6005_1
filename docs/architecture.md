# Αρχιτεκτονικη Συστηματος

## Επισκοπηση

Το project υλοποιει ενα νευρωνικο δικτυο για ταξινομηση καρδιακων παθησεων χρησιμοποιωντας το συνολο δεδομενων Cleveland Heart Disease.

### Τεχνικες Λεπτομερειες

| Στοιχειο | Περιγραφη |
|----------|-----------|
| Αρχικοποιηση Βαρων | Xavier Uniform |
| Ενεργοποιηση | ReLU |
| Κανονικοποιηση | BatchNorm + Dropout + L2 Weight Decay |
| Συναρτηση Απωλειας | CrossEntropyLoss |
| Optimizer | SGD με Momentum (η Adam) |
| LR Scheduler | ReduceLROnPlateau |
| Πρωιμη Διακοπη | Με patience 30 εποχων |

## Στρατηγικες Αξιολογησης

### Μετρικες
- Accuracy (Ακριβεια)
- Precision (Ακριβεια Θετικων)
- Recall (Ανακληση)
- F1-Score (Αρμονικος Μεσος)
- Confusion Matrix (Πινακας Συγχυσης)

### Αναλυση Γενικευσης
- Κενο Εκπαιδευσης-Επικυρωσης
- Κενο Επικυρωσης-Δοκιμης
- Ανιχνευση Υπερπροσαρμογης/Υποπροσαρμογης
