# CN6005 - Εργασια Τεχνητης Νοημοσυνης

Νευρωνικο Δικτυο για Ταξινομηση Καρδιακων Παθησεων

## Επισκοπηση

Ταξινομητης Multi-layer Perceptron (MLP) για προβλεψη σοβαροτητας καρδιακων παθησεων χρησιμοποιωντας το συνολο δεδομενων Cleveland Heart Disease απο το UCI ML Repository.

- **Δειγματα:** 303
- **Χαρακτηριστικα:** 13 κλινικα χαρακτηριστικα
- **Κλασεις:** 5 (0 = καμια παθηση, 1-4 = αυξανομενη σοβαροτητα)

## Εγκατασταση

```bash
# Δημιουργια εικονικου περιβαλλοντος
python3 -m venv venv
source venv/bin/activate

# Εγκατασταση εξαρτησεων
pip install -r requirements.txt
```

## Χρηση

### Διαδραστικο CLI

```bash
python -m app.cli
```

Χρησιμοποιηστε τα βελακια για πλοηγηση στο μενου και επιλογη.

### Γρηγορη Εκπαιδευση

```bash
python -m app.main
```

## Δομη Εργου

```
app/
  cli.py              # Διαδραστικο CLI
  main.py             # Κυριο σημειο εισοδου
  config/             # Ρυθμισεις
  data/               # Προεπεξεργασια δεδομενων
  models/             # Νευρωνικο δικτυο
  training/           # Εκπαιδευση & αξιολογηση
  visualization/      # Γραφηματα
  lib/                # Βοηθητικες συναρτησεις
  tests/              # Δοκιμες μοναδας
  outputs/            # Αποτελεσματα
docs/                 # Τεκμηριωση
```

## Νευρωνικο Δικτυο

```
Εισοδος (13) -> Κρυφο (64) -> Κρυφο (32) -> Κρυφο (16) -> Εξοδος (5)
```

Καθε κρυφο επιπεδο περιλαμβανει:
- Batch Normalization
- Ενεργοποιηση ReLU
- Dropout (0.3)

## Αρχεια Εξοδου

Μετα την εκπαιδευση, ελεγξτε το `app/outputs/`:
- `training_curve.png` - Καμπυλες απωλειας/ακριβειας
- `confusion_matrix.png` - Προβλεψεις vs πραγματικες τιμες
- `classification_report.txt` - Αναλυτικες μετρικες
